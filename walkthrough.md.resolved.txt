## Setup Instructions

### Prerequisites
- **Python 3.10** — https://www.python.org/downloads/release/python-3100/
- **Node.js 18+** — https://nodejs.org/

### Backend
```powershell
cd C:\Users\0\.gemini\antigravity\scratch\rag-chatbot\backend

python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt

copy .env.example .env
# Edit .env or use the Settings tab in the UI

uvicorn main:app --reload --port 8000
```

### Frontend
```powershell
cd C:\Users\0\.gemini\antigravity\scratch\rag-chatbot\frontend
npm install
npm run dev
```

Open **http://localhost:5173**

## Usage Flow

1. **⚙️ Settings tab** → enter LLM API URL, token, model → **Save**
2. **Sidebar** → drag & drop a PDF/TXT/DOCX → wait for indexing
3. **💬 Chat tab** → ask questions about your documents

---

## Key Design Choices

- **Dynamic LLM config** — URL/token reloaded on every request, changes apply instantly
- **Local embeddings** — `all-MiniLM-L6-v2` runs on-device, no embedding API cost
- **OpenAI-compatible** — works with OpenAI, Ollama, LM Studio, Groq, Azure, etc.
- **ChromaDB** — zero-infra persistent vector store (local folder `chroma_db/`)
